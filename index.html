Démarrer

Une seule ligne à changer.

Drop-in du SDK OpenAI officiel. Pointez la baseURL vers Infera, utilisez votre clé, et gardez tout votre code.

example.ts 
// Drop-in OpenAI SDK — only baseURL changes
import OpenAI from "openai";

const client = new OpenAI({
  baseURL: "https://api.infera.eu/v1",  // ← here
  apiKey : process.env.INFERA_API_KEY,
});

const res = await client.chat.completions.create({
  model: "mistral-large",
  messages: [{ role: "user", content: "Bonjour, Europe." }],
});
