Intégration

Une variable d'environnement.

python · node · curl 
# Python
from openai import OpenAI
client = OpenAI(
  base_url="https://api.infera.eu/v1",
  api_key=os.environ["INFERA_API_KEY"],
)

# Node.js
import OpenAI from "openai";
const client = new OpenAI({ baseURL: "https://api.infera.eu/v1" });

# curl — identique à OpenAI
$ curl https://api.infera.eu/v1/chat/completions \
    -H "Authorization: Bearer $INFERA_API_KEY" \
    -H "Content-Type: application/json" \
    -d '{"model":"mistral-large","messages":[{"role":"user","content":"Bonjour"}]}'
Modèles

Open-weight, au plus près de l'état de l'art.

Catalogue mis à jour en continu. Précisions quantisées disponibles pour chaque modèle.

MISTRAL

mistral-large · 8×22B

Mixture of experts, contexte 128k, multilingue natif. Référence sur raisonnement FR.

META

llama-3.3 · 70B

Tool use natif, function calling stable. Excellent sur tâches code et analyse.

DEEPSEEK

deepseek-v3 · 671B

MoE extrême, raisonnement long. Idéal pour agents complexes et code.

ALIBABA

qwen-2.5 · 72B

Contexte 1M, excellent français, bonne vision. Option multilingue robuste.

EMBEDDINGS

nomic-embed · v1.5

768-dim, licence Apache, très performant sur retrieval multilingue.

VISION

pixtral-large · 124B

Vision + texte, contexte 128k. Parfait pour OCR et compréhension de doc.
