Installation

Une ligne pour rejoindre le réseau.

terminal 
# Linux / macOS — agent HIA + LocalAI
$ curl -fsSL https://install.infera.eu | sh

# Ou via Docker
$ docker run -d --gpus all --name infera \
    -e INFERA_NODE_NAME="lab-fr-01" \
    -e INFERA_TIER="public" \
    -v /var/lib/infera:/data \
    infera/agent:latest
